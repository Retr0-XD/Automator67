## Architecture Decision: Optional Authentication

**Decision**: Authentication is completely optional to align with user-owned infrastructure model.

- **Local Mode**: No authentication, works offline, data in localStorage
- **Cloud Mode**: GitHub OAuth via user's own OAuth app + backend
- **No centralized services**: User deploys their own backend if sync is needed
